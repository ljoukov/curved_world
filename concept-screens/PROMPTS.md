# Concept screen generation prompts

All screens were generated with the built-in `image_gen` tool using the `ui-mockup` use case. For each concept, screen 1 was used as the visual and character reference for screen 2.

## Curved World — 01 lesson

```text
Use case: ui-mockup
Asset type: polished landscape desktop web-app screen for a one-minute hackathon demo
Primary request: Design the main interactive lesson screen for “Curved World,” an online geometry tutor from a hyperbolic universe.
Scene/backdrop: A shippable browser app interface styled as a whimsical retro-futurist school laboratory, dark indigo and warm ivory surfaces with coral, mint, amber, and cyan accents; subtle paper grain and luminous instrument-panel details.
Subject: The dominant workspace is an accurate-looking Poincaré disk: a large circular hyperbolic geometry canvas containing one triangle whose sides are curved geodesic arcs meeting the disk boundary orthogonally. Place three small angle markers and a compact result readout showing exactly “ANGLE SUM 143°”. To the right, a clear video-call tutor panel contains Professor Bent, a memorable friendly nonhuman geometry professor: an ageless teal-skinned humanoid with a crescent-curved head silhouette, round amber spectacles, swept white hair, burgundy academic robe with gold piping, mint bow tie, kind but slightly alarmed expression. Include a small live indicator and a speech bubble containing exactly “Excellent. Plenty of angle left over.”
Interface hierarchy: Slim top bar with the exact product name “CURVED WORLD”; compact lesson label “LESSON 01 · TRIANGLES”; large geometry canvas on the left; tutor/video panel on the right; bottom tool rail with simple icon buttons for Point, Geodesic, Triangle, Undo, and a primary button labeled exactly “CHECK WORK”.
Style/medium: High-fidelity 2D product UI mockup, precise modern layout, bespoke editorial illustration integrated into the app, restrained retro-futurist educational software aesthetic, production-ready rather than concept art.
Composition/framing: Full straight-on 16:9 desktop browser viewport, no device frame, generous spacing, crisp panels and readable practical hierarchy.
Lighting/mood: Soft luminous screen lighting, clever, warm, whimsical, intellectually strange.
Text (verbatim): “CURVED WORLD”; “LESSON 01 · TRIANGLES”; “ANGLE SUM 143°”; “Excellent. Plenty of angle left over.”; “CHECK WORK”
Constraints: Keep all listed text short, legible, and spelled exactly. Visually communicate hyperbolic geometry immediately. Keep the Professor Bent character design distinctive and reusable for a second screen. Use only sparse supporting microcopy.
Avoid: external brands, logos, watermarks, illegible text walls, phone layout, ordinary straight-sided Euclidean triangle, generic chatbot UI, photorealism, excessive decoration, visual clutter.
```
